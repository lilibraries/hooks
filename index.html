Getting Started

@lilib/hooks is a React hooks library.

Install#

npm install --save @lilib/hooks
# Or
yarn add @lilib/hooks

Example#

import React from "react";
import { useMount } from "@lilib/hooks";
function Example() {
useMount(() => {
console.log("Component is mounted");
});
...
}

Hooks#

CategoryHookDescription
LifecyclesuseMountRun effect after the component is mounted
useUpdateRun effect on the component or the deps update
useUnmountRun effect on the component unmount
useUpdateLayoutEffectIt's similar to useUpdate
useIsomorphicLayoutEffectIsomorphic layout effect
StatesuseToggleUse boolean value
useDebouncedDebounce a value
useThrottledThrottle a value
CallbacksusePersistReturns the same function no matter whether the callback param changes
useDebounceUse debounced callback
useThrottleUse throttled callback
useTimeoutDelay running callback
useIntervalLoop run callback
useRafUse requestAnimationFrame to invoke the callback before the browser repainting
useIdleInvoke the callback when the browser is idle
useNextFrameEnsure the callback is invoked after the current paiting
useAnimationUse this hook to perform JavaScript animation
RefsuseMountedReturns a ref object to track if the component is mounted
useLastestReturns a ref object whose .current property is assigned to lastest updated value
usePreviousReturns a ref object whose .current property is assigned to previous updated value
BOM and DOMuseTitleSet document title
useFaviconSet document favicon
useOnlineDetect whether the network is online
useDarkModeDetect whether the system prefers color schema is dark
usePageVisibleDetect whether the page is visible
useWindowSizeReturns window width and height
useEventListenerSets up the event listener to the target
useCookieGet or set cookie
useLocalStorageGet or set localStorage
useSessionStorageGet or set sessionStorage
UtilsuseCreateIt's similar to React.useMemo
useRerenderReturns a function that you can use to rerender the component
